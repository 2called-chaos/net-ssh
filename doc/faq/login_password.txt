To log in to a server using a password, just specify the password in the options to the start method:

<h:code lang="ruby">
require 'net/ssh'

Net::SSH.start("host.com", "bobsmith", :password => "g00d1es") do |ssh|
  # ...
end
</h:code>

Obviously hard-coding something like a password is not usually a good idea! Net::SSH comes with a Prompt module that gives you a way to prompt for passwords, which is a bit more secure:

<h:code lang="ruby">
require 'net/ssh'

include Net::SSH::Prompt
passwd = prompt("Give the passphrase, peasant, or taste steel:", false)

Net::SSH.start("host.com", "bobsmith", :password => passwd) do |ssh|
  # ...
end
</h:code>
